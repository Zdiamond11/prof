# Профсоюзный веб-портал

## Описание проекта
Веб-портал для профсоюзной организации, обеспечивающий внутреннюю коммуникацию, документооборот, учет членов профсоюза и финансовую деятельность.

### Функциональные модули:
1. Аутентификация и авторизация с 2FA
2. Лента новостей
3. Справочник членов профсоюза
4. Протоколы собраний и мотивированные мнения
5. Голосования (включая кворум председателей)
6. Финансовый модуль (фонд взаимопомощи)
7. Система рассылок
8. Отчетность и дашборды

## Технический стек
- Django (последняя LTS версия)
- PostgreSQL
- Python 3.11+
- HTML, CSS, JavaScript, Bootstrap 5
- Chart.js, python-docx
- Docker

## Установка и запуск
1. Установите зависимости: `pip install -r requirements.txt`
2. Настройте базу данных в `settings.py`
3. Выполните миграции: `python manage.py migrate`
4. Запустите сервер: `python manage.py runserver`
